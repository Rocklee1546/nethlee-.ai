# 🤖 NethLee AI

**NethLee AI** is an intelligent website Question & Answer chatbot that leverages **Web Scraping** and **Natural Language Processing (NLP)** to extract, process, and retrieve information from websites. Users can enter any website URL, and the chatbot will scrape the content, process it using NLP techniques, and answer questions based on the extracted information.

---

## 🚀 Features

* 🌐 Website content scraping using BeautifulSoup
* 🤖 Intelligent Question & Answer chatbot
* 🧠 NLP-based text preprocessing and semantic search
* 🔍 TF-IDF and Sentence Embeddings for information retrieval
* 📊 Interactive analytics dashboard
* 💬 Modern chatbot interface
* 📜 Chat history management
* 📈 NLP statistics and visualizations
* ⚡ Fast response using FAISS vector search
* 🎨 Responsive and modern UI with Tailwind CSS

## ⚙️ How It Works

1. Enter a website URL.
2. The application scrapes and extracts website content.
3. HTML is cleaned and converted into plain text.
4. NLP preprocessing is performed.
5. Content is divided into searchable chunks.
6. Sentence embeddings are generated and stored in FAISS.
7. Users ask questions in natural language.
8. The chatbot retrieves the most relevant information and returns an accurate answer with source references.
